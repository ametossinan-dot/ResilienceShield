# ResilienceShield — Middleware de résilience réseau

Projet de fin d'études — Bénin Hub Technologie 2025

## Auteurs
- AHOUDJI Francis (Mini ERP)
- AMETOSSINA Norberto (Middleware)

## Description
Middleware universel de continuité d'activité pour PME.
Permet aux logiciels métiers de fonctionner sans interruption
même en cas de panne réseau.

## Technologies
- Go 1.22
- SQLite
- API REST
- Dashboard HTML/CSS/JS

## Lancement
```bash
go run main.go
```

Dashboard : http://localhost:8080/admin/
